Using CVS in the IDE

See Also

The IDE's built-in CVS (Concurrent Versioning System) client provides versioning functionality without requiring installation of any additional software on your computer. You can also use CVS from the command line if you have a CVS executable installed on your system.

The IDE provides two options for working with CVS:

  • The built-in CVS client. The IDE contains a CVS client written in the Java programming language that provides a direct protocol connection to CVS repositories, enabling you to use CVS without installing a separate CVS executable on your system. The IDE's built-in CVS client supports only password server (pserver) connections and does not support the edit, watch, or lock commands.
  • The CVS profile. The IDE contains a profile of CVS commands and variables that enables the IDE to interact with the CVS command-line executable. The CVS command-line executable must be installed on your system to use this module.

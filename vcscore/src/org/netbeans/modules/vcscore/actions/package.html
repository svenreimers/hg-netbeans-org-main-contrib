This package integrates VCS support with actions UI.

VCS Actions Package

There are various command actions: Add, Diff, Commit, History, Remove and Update that for toolbar. CommandMenu and VcsGlobalCommandAction action constructs context menu from {@link ComamndTree} descriptors. There are also few non command related actions: group management, opening revision explorer, versioning manager, versioning output. There are inproperly placed they should go to repective packages,

{@link CommandMenu} class describes menu invoked command lifetime (without a phase hidden in command.execute()). It's described in {@link org.netbeans.modules.vcscore.commands} package.

Code Template Tools

Features

  • Interactively create, modify or delete a code template
  • Interactively insert a code template
  • Cut, Copy and Paste to Register 0 through 9 and A- Z. The registers are implemented as Code Templates with name REG_ + registerName

Ctrl+J R X is the shortcut for Cut to Register action. Ctrl+J R C is the shortcut for Copy to Register action. Ctrl+J R V is the shortcut for Paste from Register action

How to Use

To create a Code Template from selection in the editor

  • Select text in editor.
  • Invoke "Create Template..." action
  • Name the template and insert any parameters using the ${parameter-name} syntax
  • Supports additonal parameter ${selection}. If the text editor had any selection, that selection will be inserted at the location of ${selection} parameter. Use ${selection editable=false} to prevent editing of inserted text
  • Click on Save to save the Code Template

To insert a Code Template

  • Invoke "Show Templates ..." action
  • Select a template from the list and preview it
  • Click on Insert to insert the selected Code Template.

To create a Code Template

  • Invoke "Show Templates ..." action
  • Click on New... to create a template.

To modify or delete a Code Template

  • Invoke "Show Templates ..." action
  • Select a template from the list and preview it
  • Click on Modify... to modify the selected template
  • Click on Delete... to delete the selected template

To use Registers

  • Select text in editor
  • Invoke "Cut to Register..." Ctrl+J R X, select the Register from the dialog and click Ok. The selection is cut to selected Register.
  • Invoke "Copy to Register..." Ctrl+J R C, select the Register from the dialog and click Ok. The selection is copied to selected Register.
  • Invoke "Paste from Register..." Ctrl+J R V, select the Register from the dialog and click Ok. The contents of the selected Register are pasted.

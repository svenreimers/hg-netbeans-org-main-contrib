Generating Java Files for RMI Programs

The Remote Method Invocation (RMI) module helps you write programs that use RMI technology. It contains RMI templates, which can generate most of the code needed for RMI programs. It also includes an RMI Registry browser that generates the code for RMI registry binding and lookup.

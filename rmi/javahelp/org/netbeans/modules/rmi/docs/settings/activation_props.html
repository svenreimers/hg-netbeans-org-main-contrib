RMI Activation Settings

You can customize the settings for the RMI activation system.

To modify the settings:

  1. Choose Tools  Options from the main window.
  2. Expand the node for IDE Configuration.
  3. Expand the node for Distributed Application Support.
  4. Expand the node for RMI Settings and select Activation.

The Properties pane contains the following properties:

  • Refresh Time. Time between updates of all activation system items. If the value is 0, no updates are performed.

RMI Project Settings

There are a number of project settings that affect the operation of the RMI module.

Accessing RMI Project Settings

  1. From the main window, choose Project Settings.
  2. On the Project Settings dialog, click on the "Execution Types" node and expand it. This will display subnodes, a number of which are for RMI settings.
  3. Click on the "Debugger Types" node and expand it. This will display subnodes, one of which is the "RMI Debugging" subnode.
  4. Click on the "Compiler Types" node and expand it. This will display subnodes, one of which is the "RMI Stub Compiler" subnode.
For more information see:
  • Settings for the "Execution Types" Node
  • Settings for the "Debugger Types" Node
  • Settings for the "Compiler Types" Node

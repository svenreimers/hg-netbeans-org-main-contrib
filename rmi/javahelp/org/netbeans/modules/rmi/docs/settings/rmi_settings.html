RMI Module Settings

You can customize the settings for the RMI module.

To modify the settings:

  1. Choose Tools  Options from the main window.
  2. Expand the node for Distributed Application Support and select RMI Settings.

The Properties pane contains the following properties:

  • Confirm Marking. If true, opens a dialog box asking user to confirm that selected object should be marked as an RMI object. The default value is true, meaning any JavaDataObject recognized by the IDE as a RemoteObject is automatically marked as an RMI object.
  • Default Stub Compiler. The name of the default stub compiler.
  • Detect Remote. If true, the RMI module checks for RemoteObject classes.

The Info pane contains the following properties:

  • Name. The name of the RMI Settings node.
  • Template. If true, uses this object as a template.

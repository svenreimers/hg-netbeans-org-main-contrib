Viewing Information for Running Registries

The RMI Registry Browser enables you to browse information on individual RMI registries that are running on your local machine or elsewhere on the network.

To browse a remote registry:

  1. Click the Runtime tab of the Explorer.
  2. Right-click the RMI Registry node and choose Add Registry.
  3. In the Add Registry dialog box, select the default hostname and type the name of the remote host.
  4. Select the port number and type the port number of the remote host.
  5. Click Yes to start the registry.

    A subnode that represents the new registry is added to the RMI Registry node.

  6. Right-click on any of the service nodes of the registry and choose Properties from the contextual menu to examine information for the service.

Adding an Activation Browser

The RMI Activation Browser node is located in the Runtime tab of the Explorer window. Right-clicking this node opens a contextual menu with items for refreshing the display of the subtree, mounting running systems, and displaying properties.

To add an instance of the Activation browser:

  1. Make sure the activation system is running.
  2. Right-click the Activation Browser node and select Add Activation System.
  3. In the Add Activation System dialog box, click OK to add an instance of the system with the default host and port number, localhost and port 1099.

    To use a host other than the default host, type the name of the host and its port number and click OK.

A node for the activation system is displayed under the Activation Browser node. The node designation is in the format of hostname: port_number.

Activation System Node Commands

You can perform actions on the subnode displayed within the Activation System Browser for a specific activation system. Right-click the node to open the contextual menu. The menu has the following items:

  • Add Activation Group. Registers a new activation group using the Activation Group dialog box for the group.
  • Refresh. Updates the activation system browser.
  • Shutdown. Shuts down the activation system.
  • Copy Setup Code. Generates and places code that performs the setup of the activation structure provided by the browser.
  • Delete. Removes a given activation system from the tree.
  • Properties. Displays the current values for the following properties related to the activation system and the browser:
    • Hostname. The host where the activation system is running.
    • Port. The port number of the activation system.

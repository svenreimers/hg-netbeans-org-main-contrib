Activatable Object Node Commands

You can perform actions on the node that represents activatable objects. Right-click the node to open the contextual menu. The menu contains the following items:

  • Activate Object. Activates the object.
  • Inactivate. Deactivates the object.
  • Register. Registers the object into an RMI registry.
  • Copy Setup Code. Generates and places in the clipboard code that performs the setup of the activatable object.
  • Delete. Removes the object from the tree and unregisters it from the activation system.
  • Customize. Displays the customization dialog box that you use to edit the properties of the activatable object.
  • Properties. Displays the current values for the properties of the selected activatable object.

Compiling a Server File

The last step in completing a server file is to compile all the classes related to the server, including the remote interface.

To compile the server file:

  • Right-click the node for the RMI server class and select Compile from the contextual menu.

    The remote interface is also compiled and the stub and skeleton files are generated.

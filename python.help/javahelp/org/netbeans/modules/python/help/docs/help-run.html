Running Python Applications

You can run the whole project or individual files

  1. To run the whole project, Right-Click on the project in the Project Explorer and select Run. If there is only one open project, then you can also run it using F6. However, in case multiple projects are open, F6 will run the Main Project
  2. To Run a file, click on the file name in the project explorer and select Run > Run File (F6)

Options Window: Customizing JavaFX Syntax Settings

See Also

The Syntax tab in the Fonts & Colors options window enables you to customize the different syntax categories available for the JavaFX Script language.

To customize the syntax categories for the JavaFX Script language:

  1. Choose Tools > Options from the main menu.
  2. Click the Fonts & Colors category.
  3. Click the Syntax tab and select JavaFX from the language combo box.
  4. Select a category and set the font, the foreground color, the background color, effects, and effect color to be used.
    The Preview pane displays how your selection will appear.
  5. Click OK to save your selected settings.

About Building JavaFX Script Applications

See Also 

The IDE uses an Ant build script to build your JavaFX Script applications. It generates the build script based on the options you enter in the project's Properties dialog box.

You can customize the build process by doing any of the following:

  • Enter basic options, such as classpath settings and JAR filters, in the Project Properties dialog box.
  • Override IDE-generated Ant targets or create new targets in build.xml.
  • Edit properties in project.properties to change the name and location of build output directories and files.

JavaFX projects can contain both JavaFX Script and Java source files. The OpenJFX compiler is used to compile the JavaFX Script source file and the Java compiler is used to compile Java source files. Upon a successful build of the project, a single JAR file is created to contain both the compiled JavaFX Script and Java source code.

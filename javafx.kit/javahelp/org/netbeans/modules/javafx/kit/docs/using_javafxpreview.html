Using the JavaFX Script Preview Feature

See Also 

The JavaFX Script preview feature enables you to view the changes you make to your JavaFX Script source file without having to build and run your application first. If no errors are encountered, changes you make to the current JavaFX Script source file are displayed in the canvas pane located just above the source editor pane.

  • To enable or disable the JavaFX Script preview feature, click the Preview button in the source editor's toolbar.
  • To view the current state of the active source file, click the Reset Preview button in the source editor's toolbar.

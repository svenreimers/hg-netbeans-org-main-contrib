JavaFX Project Tasks: Quick Reference

See Also

This topic describes common tasks that you can perform with the JavaFX projects. For more detailed information, click the links in the right column.

To perform this task... Follow these steps
Create a new JavaFX project
  1. Choose File > New Project (Ctrl-Shift-N).
  2. In the New Project wizard, select JavaFX in the Categories list.
  3. In the Projects List, select JavaFX Script Application and click Next.
  4. In the Name and Location panel, specify the project's name and location.
  5. Select Empty Project.
  6. Set the project as the main project and create the main class file, if so desired.
    See New JavaFX Script Application Wizard: Name and Location for more information
Import an existing JavaFX Script application
  1. Choose File > New Project (Ctrl-Shift-N).
  2. In the New Project wizard, select JavaFX in the Categories list.
  3. In the Projects List, select JavaFX Script Application and click Next.
  4. In the Name and Location panel, specify the project's name and location.
  5. Select From Sources.
  6. Click Add Folder to navigate and select the folder that contains your existing source files.
  7. Set the project as the main project and create the main class file, if so desired.
    See New JavaFX Script Application Wizard: Name and Location for more information
Add files to your project
  1. In the Projects window, select the main project node and right-click it.
  2. Select New and choose the type of file you would like to add.
  3. In the New File wizard, specify the file name, location, and package.
  4. Click Finish to create the file.
Edit files
  1. In the Projects window, select the node for your project and, if necessary, double-click it to open the node.
  2. Navigate to the file you want to edit and double-click it to open it in the source editor.

For more information on the source editor, see Editing JavaFX Source Files.

Build a project.
  • Choose Build > Build Main Project (F11) or right-click any project node and choose Build Project.
Run a project.
  • Choose Run > Run Main Project (F6) or right-click any project node and choose Run Project.
Debug a project.
  • Choose Run > Debug Main Project (Ctrl-F5) or right-click any project and choose Debug Project.
    Note: Currently, you can only debug Java source code when debuggin a JavaFX project.
Deploy a project.
  • Choose Run > Run Main Project (F6) or right-click any project node and choose Run Project.
For more deployment information, see Deploying JavaFX Script Applications.

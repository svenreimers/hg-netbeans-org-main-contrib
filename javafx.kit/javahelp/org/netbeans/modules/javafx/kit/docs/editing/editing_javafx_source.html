Editing JavaFX Script Source Files

See Also

The IDE's source editor provides the following features for editing source files written in the JavaFX Script language:

  • Syntax highlighting
  • Error detection
  • Indentation and reformatting. The IDE indents lines as you type. You can format selected lines or the entire file by pressing Alt-Shift-F.
  • Code completion. As you type, a dialog box displays the possible ways to complete the JavaFX Script expression you are typing. Navigate to the correct one and press Enter.
  • Code folding. Enables you to hide or display code blocks in the source file.
  • Preview. Using the JavaFX Script preview feature, view the effects of the edits you make to the current source file.

To open a JavaFX source file in the Source Editor:

  • Double-click the JavaFX source file's node in the Projects window.

To enable or disable syntax highlighting and error detection:

  • Click the Error Detection and Syntax Highlighting button on the editor toolbar.

To enable or disable the preview feature:

  • Click the Preview button on the source editor's toolbar.

The editor toolbar also provides buttons for other editor commands available for the currently displayed source file. For a description of these buttons, see Editor Toolbar.

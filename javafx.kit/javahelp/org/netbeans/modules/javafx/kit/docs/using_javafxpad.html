Using the JavaFX Preview Pad

See Also 

The JavaFX preview pad allows you to view the changes you make to your JavaFX source file without having to build and run your application first. If no errors are encountered, changes you make to in the JavaFX source editor are displayed in the canvas pane located just above the source editor pane.

  • To enable or disable the JavaFX preview pad, click the Preview button in the editor toolbar.
  • To view the current state of the active source file, click the Reset Preview button restart source code from editor and shows the result on the preview panel.

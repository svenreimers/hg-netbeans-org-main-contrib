Working with JavaFX Projects

The following table shows how to work with JavaFX projects.

Create and configure project
  1. Create the JavaFX project by choosing File > New Project (Ctrl-Shift-N) and select the JavaFX Application template.
  2. Follow the steps in the remainder of the wizard.
  3. Configure the classpath for the project.


Build project

  • Choose Build > Build Main Project or right-click any project and choose Build Project.

Run project
  1. If necessary, set the main class and arguments for the project.
  2. Choose Run > Run Main Project or right-click any project and choose Run Project.

Debug Java code in a JavaFX project

Note: Currently, only Java code can be debugged within a JavaFX project.

  1. Set breakpoints or watches in your code.
  2. Choose Run > Debug Main Project or right-click any project and choose Debug Project. You can also attach the debugger to a running process.
  3. Step through the program execution.
  4. Choose Run > Finish Debugging Session when you are finished debugging.

Profile project

Note: Currently, profiling of a JavaFX project can be done at the project level only.

  • Choose Profile > Profile Main Project or right-click any project and select Profile.

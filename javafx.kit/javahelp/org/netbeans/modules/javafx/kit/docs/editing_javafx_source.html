Editing JavaFX Source Files

See Also

The IDE's Source Editor provides the following features for editing JavaFX source files:

  • Syntax highlighting
  • Error detection
  • Indentation and reformatting. The IDE indents lines as you type. You can format selected lines or the entire file by pressing Alt-Shift-F.
  • Code completion. As you type, a dialog box displays the possible ways to complete the JavaFX expression you are typing. Navigate to the correct one and press Enter.
  • Code folding. Enables you to hide or display code blocks in the source file.
  • Preview. Using the JavaFX canvas pad, view the effects of the edits you make in the current JavaFX source file.

The editor toolbar also provides buttons for some of the other editor commands available for the currently displayed Java file. For a description of those buttons, see Editor Toolbar.

To open a JavaFX source file in the Source Editor:

  • Double-click the JavaFX source file's node in the Projects window.

To enable or disable syntax highlighting and error detection:

  • Click the Error Detection and Syntax Highlighting button on the editor toolbar.

To enable or disable the preview pad:

  • Click the Preview button on the editor toolbar.

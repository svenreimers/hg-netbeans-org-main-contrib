Setting the Target JDK in a JavaFX Project

See Also 

By default, every new JavaFX project uses the JavaFX SDK platform as the default platform the IDE uses for compilation, execution, and debugging of JavaFX applications. This platform is based on the bundled JavaFX libraries and the default Java SE platform (JDK) that is bundled with the IDE. You can view your IDE's JDK version by choosing Help > About and clicking the Detail tab. (On a MacOS platform, choose NetBeans > About NetBeans). The JDK version is listed in the Java field.

  • You can run the IDE with a different JDK version by starting the IDE with the --jdkhome  jdk-home-dir switch on the command line or in your IDE-HOME/etc/netbeans.conf file. For more information, see IDE Startup Parameters.

To set the default platform for a JavaFX project:

  1. Right-click the project's root node in the Projects window and choose Properties.
  2. In the Project Properties dialog box, select the Libraries node in the left pane.
  3. Choose the desired JavaFX platform in the JavaFX Platform combo box.
  4. Select the desired platform in the Platforms pane on the left and click Close.

    Switching the target Platform for a JavaFX project does the following:

    • Offers the new target platform's classes for code completion.
    • If available, displays the target JDK's source code and Javadoc documentation.
    • Uses the target SDK's executables (javafxc and javafxrt) to compile and execute your application.
    • Compiles your source code against the target SDK's libraries.

To register additional JavaFX platforms with the IDE:

  1. From the Java Platform Manager, click the Add Platform button.
  2. In the Add Java Platform wizard, select Java JavaFX Platform and click Next.
  3. Specify the Platform Name you want assigned and the Java platform to use. Use the Browse button to navigate to the JDK/JRE you want to use.
  4. Specify the folder that contains the necessary JavaFX libraries to use with the new JavaFX platform.
  5. Click Finish.
    The new platform is added to the list of available JavaFX platforms.
  6. Click Close to dismiss the Java Platform Manager dialog.

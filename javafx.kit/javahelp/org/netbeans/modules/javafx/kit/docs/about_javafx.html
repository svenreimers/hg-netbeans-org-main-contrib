Support for the JavaFX Script language in NetBeans has the following features:

Advanced Source Code Editor

The language-aware editor indents and completes your source code written in the JavaFX Script language. The editor also performs semantic syntax highlighting, live parsing of your code, matches words and brackets, marks errors and occurrences, and has code folding capability. It has a toolbar that includes controls to enable or disable the preview, syntax highlighting, or error detection features. For more information, see Editing JavaFX Script Source Files

Preview Feature

The JavaFX Script plugin include a preview feature that allows you to view the effects of the changes you make to your source code without having to build and run your application each time you make any changes. If no errors are encountered, whatever changes you make in the source editor will be displayed in the canvas pane located just above the source editor pane.

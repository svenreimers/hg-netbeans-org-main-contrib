JavaFX Script support in NetBeans has the following features:

Advanced Source Code Editor

The JavaFX Script language-aware editor indents and completes your JavaFX Script code and performs semantic syntax highlighting. The editor also does live parsing of your code, matches words and brackets, marks errors and occurrences, and has code folding capability. It has a toolbar that includes controls to enable or disable the previewer, the syntax highlighting feature, and error detection feature. For more information, see Editing JavaFX Source Files

JavaFX Preview Pad

The JavaFX Preview Pad allows you to view the effects of the changes you make to your source code without having to build and run your application each time you make any changes. If no errors are encountered, whatever changes you make in the source editor will be displayed in the canvas pane located just above the source editor pane.

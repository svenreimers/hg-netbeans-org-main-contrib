Deploying JavaFX Script Applications

See Also 

You can deploy your JavaFX Script application as a standalone application, as an application enabled with Java Web Start, or as an applet viewable in a platform-independent applet viewer.

To run a standalone application from the IDE:

  1. In the Projects window, select the main node for your project and right-click it.
  2. Select Run Project.

To run a standalone application from the command line:

  1. Navigate to the project's dist folder.
  2. Type the following at the command prompt:
           javafx -jar application_jar_name.jar  main_JavaFX_class_name.Main

To build and run a JNLP application using the IDE:

  1. In the Projects window, right-click your project's node and select Properties.
    Alternatively, you can select the project in the Projects window and select File > "<JavaFX_project_name>" Properties from the main menu.
  2. In Project Properties dialog, select the Application > WebStart category.
  3. Select the Enable Web Start and Compress JARs with Pack 200 checkboxes on the right-hand pane.
  4. In the Categories pane, select the Run cateogry and select the Enable Run with Java Web Start checkbox.
  5. Click OK.
  6. In the Projects window, right-click the project's node again and select Run Project.
    The IDE builds the project first, adds the following files to your project's folders, and then executes the project.
    • jnlp-impl.xml in nbproject folder - contains the set of ANT targets needed to generate the JNLP file, and all other related files.
    • master.jnlp file in your project's main folder - used to create the launhc.jnlp file.
    • launch.jnlp and launch.html files in the dist folder - the launch.html file is linked to launch.jnlp file

To run a standalone application with Java Web Start:

Do one of the following after you build the JNLP application for your JavaFX project:
  • Navigate to the project's dist folder and double-click the launch.jnlp file.
  • Navigate to the project's dist folder and type the following at the command prompt:
           javaws launch.jnlp

To create a JavaFX Script applet using the IDE:

Use either of the following sets of steps to create a new JavaFX applet:

  1. In the Projects window, right-click your project's main node and select New > JavaFX Applet.
  2. Edit the new JavaFX applet file to include your applet code and save it.

    -or-
  1. Select the File > New File from the main menu of the IDE.
    The New File wizard appears.
  2. In the Choose File Type page, select JavaFX category and the JavaFX Applet file type.
  3. Specify the name and location information.
  4. Click Finish.
  5. Edit the new JavaFX applet file to include your applet code and save it.

To run a JavaFX Script applet:

  1. In the Projects window or Files window, right-click the applet class for your project.
  2. Select Run File from the pop-up menu.

Adding a Service Provider

The package that contains the service provider (typically a .jar or .zip file) must be in the IDE's /lib/ext directory.

To add a service provider:

  1. In the Runtime tab of the Explorer window, expand the JNDI node.
  2. Right-click the Providers node and choose New Provider from the contextual menu.
  3. In the New Provider dialog box, complete the fields with values that identify the new provider, and click OK.

    The new provider is represented as a subnode of the Providers node.

Specifying JNDI Settings

You can customize JNDI by editing it properties.

  1. From the IDE main window, choose Tools  Options .
  2. In the Options window, expand the IDE Configuration node, and then expand the Distributed and Embedded Tools node.
  3. Select the JNDI Support node.
  4. Click the value for the Time Out property and type a new integer value.

    The Time Out property represents the connection timeout in milliseconds.

JPDA (Java) Debugger Call Stack View Enhancements

Features

This module enhances the Call Stack View of JPDA (Java) Debugger by adding the following columns:
  • Modifiers - this columns displays the modifiers of the method associated with the stack frame.
  • Method - this columns displays the name, parameters and return type of the method associated with the stack frame.
  • Declaring Class - this column displays the class declaring the method associated with the stack frame.
  • this Class - this column displays the class of this variable associated with the non-static stack frame. Note: The class this of may be different from the Declaring class.
  • Location Path - when available this column displays the full path of the location of the method associated with the stack frame.
The call stack view shows the custom icons (similar to Navigator window) for constructors and methods with public, protected, private, (package), and static qualifiers. This behaviour can be turned off by setting the system property - -Dorg.netbeans.modules.debugger.callstackviewenhancements.dontShowCustomIcons=true

How to use

Simply debug any Java application. You should see the above additional information in the Call Stack view.

The following popup actions are supported:

  • Go to type of this - Go to source file of the class of the this variable associated with the stack frame if it is different from the class that declares the method associated with the stack frame.
  • A set of Go to type [local variable type] - Go to (distinct) declared and actual types of local variables of the method associated with the stack frame.
  • Show Classes... - Show the table of all classes loaded in the debugee. Each row in Classes table shows Class name, Package name and the value returned by the toString() method of class's ClassLoader. This can be useful for debugging applications such as App Servers, IDE Frameworks (e.g. Netbeans itself), that employ a complex multi-ClassLoader architecture. This view helps you determine which ClassLoader loaded the class.

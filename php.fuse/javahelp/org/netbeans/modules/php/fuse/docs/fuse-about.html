About FUSE Support

FUSE plugin into NetBeans IDE provides basic support for .

The Project Creation wizard enables setting up new PHP projects with imported FUSE framework and setup of general config FUSE file.

There are these improvements for NetBeans IDE:

  • Generating FUSE classes for the NetBeans projects
  • Associating FUSE template classes .tmpl
  • Generating FUSE project with database setup
  • Controller -> Template variable code completion in the editor

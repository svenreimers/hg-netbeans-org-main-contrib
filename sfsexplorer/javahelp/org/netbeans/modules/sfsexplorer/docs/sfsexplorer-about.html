About System FileSystem and META-INF services Browser

This modules displays the System FileSystem and META-INF services in a window. To show the System FileSystem Browser select Window:System FileSystem and META-INF services Browser menu item.

The special features of System FileSystem and META-INF services Browser are:

  • the attributes of the file or the folder as a child node
  • the popup menu shows the actions that show the documentation related to the node or it's parents in browser. The documentation may be API, Architecture Descriptions, FAQ, File format documents etc.
  • the registered META-INF services

The popup actions are configured using the FolderLookup mechanism under the special folder called org.netbeans.modules.sfsexplorer in the System FileSystem. The actions can be generic or configured based on the platform cluster name in the netbeans.home directory. That way the actions can be made sensitive to the NetBeans platform you are running.

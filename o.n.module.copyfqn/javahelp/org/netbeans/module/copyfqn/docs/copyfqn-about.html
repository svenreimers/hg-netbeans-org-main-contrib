Copy FQN

This module provides the following action:

ActionKey bindingIcon
Copy FQNCtrl+J Ctrl+C

The Copy FQN action copies the fully qualified name of the Java Class under the caret in the editor or the Java Class of selected node in the Projects/Files windows to the OS clipboard. For example with the following code:

JPa|nel panel;

with caret in the JPanel (shown by |), the Copy FQN action copies the fully qualified class name javax.swing.JPanel to the clipboard.

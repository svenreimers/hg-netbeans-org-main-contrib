Resource Bundle Lookup

All resource bundles inspected by this lookup tools must be mounted into filesystems repository !

Searched text text field is used to enter complete string, sub-string or regular expression to be searched for.

Case Sensitive checkbox tells to search engine to carry/ignore case of searched text.

Substring checkbox informs search engine that typed text is just a part of searched text.

Regular Expression check box is enabled only in JDK with regular expression support (version 1.4 and higher).
It allows to search according to given regular expression.
Case Sensitivity and Substring matching is also applied when Regular Expression enabled.

Use Resource Bundle Filter checkbox enables filtering of resource bundle names.
Filter Text can be also Case Sensitive, Substring and Regular Expression.
Only the resource bundles matching the filter text are used for search when enabled.

Search/Stop button starts/stops search engine.

Search Results table contains three columns: Resource Bundle, Key and Text.
Results can be sorted by any column by clicking its header label.
Any result row can be copied into system clipboard be selecting the best source format from its popup menu.

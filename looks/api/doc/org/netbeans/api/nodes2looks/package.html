This package provide bidirectional bridge between Looks and the Nodes API. As the package only contains one class its usage should be pretty straighfoerward. The class is a static factory which:
  1. Allows for creating Nodes for given representing object and given Look and/or LookSelector. These mathods can be used for Look based presentation in views working with nodes.
  2. Crates instace of Look which when given a Node as represented object delgates to it. This Look is intended to be used in views which work with looks when there is not possible or effective to rewrite existing presentation to Looks.

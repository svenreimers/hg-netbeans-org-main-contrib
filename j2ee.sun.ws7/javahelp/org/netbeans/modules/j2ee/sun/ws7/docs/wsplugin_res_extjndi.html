Setting Up an External JNDI Resource on the Sun Java System Web Server

See Also

To create a custom resource:

  1. Ensure the Sun Java System Web Server is running.
  2. In the Projects window, select your project's root node to make it the active node and choose File > New File (Ctrl-N).
  3. In the Categories list, select Sun Web Server 7.0 Resources, in the File Types list, select External JNDI Resource, then click Next.
  4. In the General Attributes page, do the following:
    1. (required) Enter a JNDI name for the resource.
    2. (required) Enter the external JNDI name for the resource.
    3. Enable or disable the resource by choosing True or False.
    4. Enter a description.
    5. Click Next.
  5. In the Additional Properties page, enter any properties for the resource.
  6. Click Finish.

1. Installing Launch4j Integration module

  1. Choose Tools -> Update Center

  2. Select nbextras.org Update Center (perform Howto use nbextras first if nbextras isn't listed), then click Next.

  3. Select module called „Launch4j Integration“ from the left list and add it using Add button, then click Next.

  4. Go through licence information, wait for download completion and then click Next button.

  5. Select the checkbox on the left side of module name and confirm, then finish the installation by clicking Finish button.



2: Creating Windows .exe File for NetBeans Project

Creating and editing source files is the most important function that the IDE serves. After all, that's probably what you spend most of your day doing. The IDE provides a wide range of tools that can compliment any developer's

  1. Select some NetBeans project either in Projects View or Files View.

  2. Choose Build -> Launch4j Executable menu option.

  3. Configuration dialog appears – Click Download Launch4j... button and go through Launch4j installation. If you have Launch4j already installed, continue with next step.

  4. Click Browse... button and point to the directory where you installed Launch4j product.

  5. Note that Launch4j location text field and label turns from red to black and OK button is enabled, signalizing that Launch4j install dir was recognized. Click OK

That's it! You'll see informations printed in Output window about the executable being generated, and your project with executable is ready for running and distribution. Executable is generated into the „dist“ subdirectory of the project.

Module remembers Launch4j location, so you have to specify it only the very first time or when Launch4j installation directory changes.



3: Customizing .exe Configuration

Launch4j product (http://launch4j.sourceforge.net) allows you to specify various interesting properties of created Windows exe, such as application icon or splash screen (see Launch4j homepage for whole list of features). These properties are stored in xml config file and can be altered either by direct editing of xml config file or using UI contained in Launch4j product.

NetBeans Launch4j integration module generates Launch4j config file for you, into <NB_PROJECT_DIR>/nbproject/launch4jconfig.xml file. Config file is regenerated smartly, only core options are overwritten by the module, others are left untouched. See comment contained in config file for details.

So, to give your exe an icon or splash screen or other properties, either edit <NB_PROJECT_DIR>/nbproject/launch4jconfig.xml directly or edit it with Launch4j UI.

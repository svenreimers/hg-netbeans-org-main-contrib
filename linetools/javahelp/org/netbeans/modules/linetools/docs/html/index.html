The caret position, the selection and the order of start and end of the selection is maintained when moving/copying the lines.

The Move/Copy Selection/Line Up/Down are now part of standard NetBeans editor.

Char Actions

You can use the Char actions to either deleting text (Delete key), or modifying the text - uppercase (Alt+u u) or lowercase (Alt+u l). These actions work somewhat like the t, T, f and F commands of Vi. You can type a numeric prefix nn, to find the nnth occurence of the char. If you are trying to operate on a digit you have to type ENTER key.

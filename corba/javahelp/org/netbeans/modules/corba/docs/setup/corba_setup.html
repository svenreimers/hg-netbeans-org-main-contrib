CORBA Module Setup

The CORBA module interacts with serveral pieces of software that are supplied by the ORB vendor, such as the IDL compiler, the ORB runtime, and the naming service. You need to configure the IDE so that it can interact properly with this software. The configuation tasks are:

Supported ORB Products
Specifying an ORB
Setting up the IDL Compiler
Accessing ORB Runtime Classes
Setting up the Naming Service Browser
Setting up the Interface Repository Browser

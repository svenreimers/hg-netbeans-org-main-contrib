CORBA Module Setup

The CORBA module interacts with several pieces of software that are supplied by the ORB vendor, such as the IDL compiler, the ORB runtime, and the naming service. You need to configure the IDE so that it can interact properly with this software. The configuration tasks are:

Specifying an ORB
Setting up the IDL Compiler
Accessing ORB Runtime Classes
Setting up the Naming Service Browser
Setting up the Interface Repository Browser

Supported ORB Products

The CORBA module can generate code for use with the following ORB products:
  • OrbixWeb 3.2 for Java
  • Orbix 2000 for Java
  • VisiBroker for Java 3.4
  • VisiBroker 4.x
  • JDK 1.2 ORB
  • JDK 1.3 ORB
  • J2EE ORB

The module can also generate code for use with other ORB products. However work with other ORB products is not supported.

To generate code for use with a particular ORB, the ORB itself and the ORB supplier's IDL compiler must be installed on the development machine. The other "setup" topics explain how to set up the IDE for the ORB product you are using.

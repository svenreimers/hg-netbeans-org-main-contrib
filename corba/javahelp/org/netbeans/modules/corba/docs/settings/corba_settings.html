CORBA Project Settings

There are a number of project settings that affect the operation of the CORBA module. There are two groups of settings:
  • General CORBA settings, which apply to all operations with the CORBA module.
  • ORB product settings, which apply when you are generating source files for use with a particular ORB product.

Accessing CORBA Project Settings

  1. From the main window, choose Project Settings.
  2. On the Project Settings dialog, click on the "CORBA Settings" node. This will display the general settings in the right half of the dialog. These settings apply to all operations with the CORBA module.
  3. Click on the CORBA Settings node to expand it and see its subnodes.
  4. Notice that there is a subnode for each of ORB product supported by the CORBA module. Click on a subnode. This will display the settings that are applied when you use one ORB product in the right half of the dialog. These settings control how source files are generated when you specify the ORB product represented by the node.
For more information see:
  • Settings for the "CORBA" Node

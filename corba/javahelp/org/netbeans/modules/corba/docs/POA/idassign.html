id Assignment Policy

Each POA is associated with a combination of policy values that you cannot modify after POA creation. Multiple POAs in a single server can have different policy combinations.

To change POA policies:

  1. Click the POA Policies tab.
  2. Select an alternative value for any of the following policies:
    • Id Assignment
    • Id uniqueness
    • Implicit activation
    • Lifespan
    • Request Processing
    • Servant Retention
    • Thread

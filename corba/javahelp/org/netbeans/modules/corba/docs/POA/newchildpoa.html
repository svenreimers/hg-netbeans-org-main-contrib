Specifying New Child POAs and Removing POAs

You can specify new POAs underneath the root POA.

  1. Enter the name of the child POA in the POA Name text field.
  2. Enter the name of the variable you want to use for the child POA in the Variable text field.
  3. Click the Manager drop down to either use the existing POA manager or to specify a new manager for the child POA..
  4. Select OK. The IDE generates the appropriate source code and places it into a dedicated guarded block within the server's main method.

To remove a POA:

  1. You can remove a POA from the POA hierarchy by selecting Delete on a POA node in the in the Explorer.
  2. If any child POAs are present,a confirmation dialog appears to inform you that all child POAs are deleted. A POA is always removed with its child elements

Displaying and Changing POA Properties

You can display and change the properties of a POA by using a property sheet.

To change a POA property sheet:

  1. Right-click the POA node in the Filesystems window, and choose Properties.
  2. Click the field you want to change. Select the ellipsis and enter the new information in the text box.

Displayed fields you can change include:

  • POA Manager. The POA Manager object that activates this POA.
  • POAName. The name of this POA.
  • POA Variable. The variable information for this POA.

Changing the ORB on a Root POA

The Root POA has an additional tab for an additional property: the ORB that you used to create a POA hierarchy under the root. You should exercise caution in changing ORBs in this manner, as ORB hierarchies can differ.

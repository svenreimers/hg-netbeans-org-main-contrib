Interface Repository Browser Commands

You use the Interface Repository browser by right-clicking on a CORBA Interface Repository subnode and choosing Properties. The commands displayed in the contextual menu are specific to the interface repository subnode.

CORBA Interface Repository Node Commands

This node represents the interface repository.

Add Interface Repository

Adds the interface repository you specify.

Interface Repository Node Commands

These nodes represent the IDL interfaces.

Remove Interface Repository

Deletes the selected Interface Repository browser.

Copy Code

Copies the content of a repository to the system clipboard. You can paste this code into a class file

Refresh

Refresh the selected node and all of its subnodes.

Properties

Displays the properties of the selected interface repository node.

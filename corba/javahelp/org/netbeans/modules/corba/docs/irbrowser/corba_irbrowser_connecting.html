Starting the Interface Repository Browser

The Interface Repository Browser enables you to view the contents of a running interface repository. The IDL interfaces are represented by subnodes of the browser node. You can select a node and have the browser copy the interface into your IDE flesystem.

To start the Interface Repository Browser and view a naming service, you need the interface repository's IOR.

Obtaining the Interface Repository IOR

If the interface repository is being managed by a system administrator, the interface repository IOR will probably be stored in a well-known file. Specify this file when you start the Interface Repository Browser.

If you are managing the interface repository yourself, you need to obtain an IOR when you start the interface repository from the command line. A simple way to get this IOR into the Interface Repository Browser is to cut it to the clipboard. Note that you want the string beginning with IOR and ending at the last non-blank character before Interface Repository. After copying the IOR to the clipboard you can either:

  • Start the Interface Repository Browser and paste the IOR from the clipboard to the Interface Repository Browser's startup dialog.
  • Paste the IOR from the command shell into a text editor, and save it to a text file. You can then start the Interface Repository Browser and specify this text file in the startup dialog.

Starting the Interface Repository Browser

To use the Interface Repository Browser with the interface repository process that you started earlier:
  1. Switch to the Runtime tab of the Explorer Window.
  2. Right-click the CORBA Interface Repository node and choose Add Interface Repository from the context menu. A dialog will appear which allows you to tell the naming service browser how to find the service.
  3. Enter a name to identify the interface repository in the IDE.
  4. There are two ways to supply the interface repository IOR:
    • If you copied the IOR to a file, type the URL for the file (using file:///<path to your file> ) into the URL field.
    • Paste the IOR that you cut from the command line into the IOR field.
  5. Click OK to close the dialog. A new node, with the name you specified, will appear under the CORBA Interface Repository Node. The Interface Repository browser is now communicating with the interface repository process you specified.

Generating IDL Files

You can create an IDL file in one of the following ways:

  • Selecting the Empty template, which takes you through the Design IDL pane in the New From Template wizard to create the IDL file.
  • Importing an IDL file from the Interface Repository.
  • Selecting the Simple template, which creates an IDL file containing an interface declaration. You must add the rest of the file's content manually in the Source Editor.

To generate an IDL file through the wizard:

  1. Choose File New to open the New From Template wizard.
  2. Expand the CORBA node and select Empty. Click Next to proceed.
  3. Type the name of your IDL file in the Name text field and a destination package name in the Package text field. Click Next to proceed.
  4. Select IDL Wizard as the means by which you want to generate an IDL file.

    Selecting IDL Wizard creates a new IDL file with an IDL editor while selecting Interface Repository imports an existing IDL from the Interface Repository (IR).

  5. In the Design IDL pane, define the contents of your IDL file.
    1. Select an operation from the combo box and click Create.
    2. Enter the appropriate information for the operation in the dialog box that opens and click OK.
  6. Continue adding operations as needed. Use the Remove, Edit, Up, and Down buttons to edit the context hierarchy of your IDL file. When done, click Next.
  7. Click Continue with CORBA Wizard to continue developing the CORBA application or click Finish to create the IDL file.

To import an IDL from the Interface Repository:

  1. In the Design IDL panel, click Add IR.
  2. Provide the name of the Interface Repository you are importing the IDL file from, and enter either the URL of the repository or the IOR string. The string information must be provided as one continuous string.
  3. Click OK in the CORBA Panel dialog box.
  4. Click Refresh to redisplay the repository hierarchy in the display pane.
  5. Click Next to proceed.
  6. Click Continue with CORBA Wizard to continue developing the CORBA application or click Finish to create the IDL file.

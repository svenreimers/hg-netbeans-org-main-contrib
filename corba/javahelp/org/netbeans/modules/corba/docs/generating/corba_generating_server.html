Generating Server-side Files

You can generate server-side files with either the CORBA wizard or the CORBA templates. Before generating any code, make sure the IDE is set up for the ORB product you want your CORBA programs to work with.

Using the CORBA Wizard:

  1. From the main window, choose Tools  CORBA Wizard to open the first page of the CORBA Wizard.
  2. On this page, you:
    • Type in the name of the IDL file on which your server program will be based. This can be either an existing file or a new file (which you create on page three of the wizard).
    • Select the package in which you want the wizard to put the files it generates.
  3. On the second page of the wizard, indicate which files the wizard should generate:
    • If you are writing a server based on existing IDL interface, check the Create Server and Create Implementation boxes.
    • If you are writing both the IDL interface and the server program, check the Create IDL,

      • Create Server and Create Implementation boxes.
    • If you are writing the IDL interface, and a server program that uses it, and a client program, check the Create IDL, Create Server, Create Implementation, and Create Client boxes.
  4. On the third page of the wizard, choose the settings that describe the implementation that you want to generate.
  5. The appearance of the wizard's fourth page depends on whether you are using existing IDL or writing new IDL (whether you checked the Create IDL box):
    • If you are using existing IDL, the wizard will let you identify the IDL. If the IDL is in a file, use the file browser to specify the file. If the IDL is in an interface repository, browse the CORBA Interface Repository node and select the correct interface. (To do this, you must have already started the CORBA vendor's interface repository and started up the CORBA Module's Interface Repository Browser.)
    • If you are writing new IDL, the wizard has buttons that add IDL with the correct syntax.
  6. On the fifth page of the wizard, review the description of the actions the wizard will take, then click the Finish button to begin code generation.

    7. The CORBA Module generates a local copy of the IDL you specified and java source files named <IDL_interface_name>Server. For example, if you specified hello.idl, the wizard will create a local copy of hello.idl and two .java files named helloServer.java and helloImpl.java.
Using the CORBA Templates:
  1. From the main window, choose File  New to open the Template Chooser.
  2. From the CORBA template category, select the ServerMain template.
  3. Use the fields on the right side of the Template Chooser to specify both a name for your server file and the package in which it should be generated.
  4. Click the Finish button.
  5. The CORBA module creates a CORBA server source file, named <Name_You_Specified>.java.
  6. In addition to generating server program source code with the template, you need to copy the IDL on which your server is based to the server program's package.

Starting the Naming Service Browser

The Naming Service Browser enables you to view the contents of a running naming service. The naming contexts and CORBA servers are represented by subnodes of the browser node. You can select a node and have the browser generate code that binds an object to the selected node or code that performs a resolve() and returns a reference to the selected node.

To start the Naming Service Browser and view a naming service, you need the naming service's IOR.

Obtaining the Naming Service IOR

If the naming service is being managed by a system administrator, the naming service IOR will probably be stored in a well-known file. Specify this file when you start the Naming Service Browser.

If you are managing the naming service yourself, you need to obtain an IOR when you start the naming service from the command line. For example, to start the JDK 1.2 ORB Naming Service:

  1. Open a command shell and navigate to the JDK installation directory. Navigate to the /bin directory.
  2. Start the naming service by typing: 
     tnameserv -ORBInitialPort 1050

    If the naming service returns an error message indicating that the port is use, you can use a different port number, but make a note of it - you will need to know it for the next procedure.

  3. The naming service returns information on the command line that is similar to the following output: 
     Initial Naming Context:
 IOR:000000000000002849444c3a6f6d672e6f72672f436f734e616d696e672f4
 e616d696e67436f6e746578743a312e30000000000100000000000000300
 00100000000000774696d2d7773000006db000000000018afabcafe00000
 002cbb35b05000000080000000000000000
 TransientNameServer: setting port for initial object references to: 1050

A simple way to get this IOR into the naming service browser is to cut it to the clipboard. You want the string beginning with IOR and ending at the last non-blank character before TransientNameServer. After copying the IOR to the clipboard you can either:

  • Start the Naming Service Browser and paste the IOR from the clipboard to the Naming Service Browser's startup dialog.
  • Paste the IOR from the command shell into a text editor, and save it to a text file. You can then start the Naming Service Browser and specify this text file in the Naming Service Browser's startup dialog.

Starting the Naming Service Browser

To use the Naming Service Browser with the naming service process that you started earlier:

  1. Switch to the Runtime tab of the Explorer.
  2. Right click the CORBA Naming Service node, and choose Bind New Context from the contextual menu. A dialog box appears that enables you to tell the naming service browser how to find the service.
  3. Enter the following: in the Name field  "JDK"; in the Kind field, "Naming."
  4. Supply the naming service IOR in one of the following ways:
    • If you copied the IOR to a file, type the URL for the file (using file:///<path to your file> ) into the URL field.
    • Paste the IOR that you cut from the command line into the IOR field.
  5. Click OK to close the dialog box. A new node, named JDK, appears under CORBA Naming Service. The Naming Service browser is now communicating with the naming service process you have specified.

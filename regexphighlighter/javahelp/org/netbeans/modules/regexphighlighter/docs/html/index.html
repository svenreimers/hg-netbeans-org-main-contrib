Highlight Regular Expressions Module

This module provides the functionality of highlighting the occurrences of matching regular expressions. It uses java.util.regexp.Pattern to match the regular expressions. It also highlights the regular expression groups in different colors. To highlight the regular expressions click on Edit: Highlight Regular Expressions menu item or toolbar button. Type the regular expression in the regular expression text field in the toolbar. Select the toolbar button to make the regular expression matching case sensitive. Select the toolbar button to highlight the regular expression groups in different colors. The is selected by default.

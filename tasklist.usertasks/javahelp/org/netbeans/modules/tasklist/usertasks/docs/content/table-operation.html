Using the Task List Table

Choosing visible columns/Sorting/Editing

Tasks have many properties, and by default only a few of these properties are shown in the task list table. You can set the sorting column by simply clicking on a table header. Clicking repeatedly will reverse the sort order and turn off the sorting. Use F2 to start editing a cell. Note that you can also reorder columns by dragging them around. You can also change the visible columns by clicking the little icon () which appears to the right of the table headers, straight above the vertical scrollbar (see image below). You may select multiple tasks and change some properties of the selected tasks at once. The view also remembers selected tasks and scrollbar positions.


Summary column

Different icons are used for undone () and done () tasks. The same icons are used in the editor if a line is associated with the task.

Effort column

Effort cells are painted in red using a bold font if the sum of the spent time and the remaining effort exceeds the planed effort.

Completed column

This columns contains the date when a particular task was completed. It will be filled automatically if you mark a task as done.

Starting/stopping a task

A task can be started by choosing "Start" from the local menu or by clicking on the icon in the local toolbar (). The small icon () marks the task that you are currently working on. Started task will remain started after a NetBeans restart. The owner of a started task will be set to the name of the current user if it was empty.

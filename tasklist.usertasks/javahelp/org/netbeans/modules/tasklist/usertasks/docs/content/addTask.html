How to Add a Task to the Source

You can associate a line in a file with a task.



There are 2 possibilities to add a task to the source:
  1. Just place the cursor at the desired position in the editor, open a task list and press the "New Task..." button from the local toolbar. Select "Context" tab and press the "Add..." button to associate a line in a text file or an URL with the task. You can also enter multiple tasks at once. Just press "Add Another" button instead of "OK".

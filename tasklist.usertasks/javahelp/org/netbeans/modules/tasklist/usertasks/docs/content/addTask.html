How to Add a Task to the Source

You can associate a line in a file with a task. The task selected in the table will also be highlighted in the editor with the yellow background:



There are 2 possibilities to add a task to the source:
  1. Just place the cursor at the desired position in the editor, open a task list and press the "New Task..." button from the local toolbar. Check the "Associated File" checkbox to associate the task with the source file.  You can also enter multiple tasks at once. Just press "Add Another" button instead of "OK".



  2. There is a special bridge module between the User Tasks module and the editor: Task List Editor Support. Install it and open the local menu over the left editor margin:

About Rectangular Edit Tools

This module installs the following actions in Edit:Rectangle Menu and Rectangle toolbar.

Action

Icon

Description
Rectangular Cut ( Ctrl+Alt+X ) Cuts the contents of rectangle defined by the selection
Rectangular Copy ( Ctrl+Alt+C ) Copies the contents of rectangle defined by the selection
Rectangular Paste ( Ctrl+Alt+V ) Pastes the contents of rectangle defined by the selection
Rectangular Delete ( Ctrl+Alt+Minus ) Deletes the contents of rectangle defined by the selection
Rectangular Clear ( Ctrl+Alt+Space ) Replace the contents of rectangle defined by the selection with spaces
Rectangular Replace ( Ctrl+Alt+R ) Replace the contents of rectangle defined by the selection with user specified string

Note: The leading column of the rectangle defined by the selection has to be less than the trailing column.
